文字列の長さ

'文字列'.length

例）

var name = 'kame';
console.log('あなたの名前は' + name.length + '文字です。');

出力結果 : あなたの名前は4文字です。

文字列の部分取り出し

最初の文字を0番目として数えていくことに注意。

1文字取り出す。

'文字列'.charAt(n)

文字列のn番目文字を返します。

複数の文字を取り出す。

'文字列'.slice(from,to)
'文字列'.substring(from,to)

文字列のfrom番目からto番目までの文字列を返す。
※toを省略した場合は残り全て

【違い】
「slice」はfromに負の値を指定すると後ろから数える。
「substring」はfromに負の値を指定しても0と同じ扱いになる。

'文字列'.substr(form,len)

文字列のfrom番目からlen文字文の文字列を返す。
※lenを省略した場合は残り全て

fromに負の値を指定すると後ろから数える。

文字列の分割し配列に

'文字列'.split(sep,limit)

文字列を sep を区切り文字として分割し、その配列を返します。
limit は配列の個数を制限します。

例）

var date = '2013/4/26';

console.log(date.split('/'));

出力結果 : ["2013", "4", "26"]

文字列を検索・置換する。

文字列の検索した位置を返す。

'文字列'.indexOf(key,form)

文字列のfrom番目から後方に検索し、最初に key が現れる位置を返します。
見つからない場合は-1を返します。　
※ formを省略した場合は0番目から検索

文字列の置換

'文字列'.replace(key,newString)

文字列内の regexp をnewStringに置換する。
