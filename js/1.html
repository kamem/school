変数・演算子

var 変数名;

【演算子（代入演算子）】

  • a = b;　a に b を代入
  • a += b;　a と b を足した値をaに代入（a = a + b）
  • a -= b;　a と b を引いた値をaに代入（a = a - b）
  • a *= b;　a と b を掛けた値をaに代入（a = a * b）
  • a /= b;　a と b を割った値をaに代入（a = a / b）
  • a %= b;　a と b を割った余りをaに代入（a = a % b）

aに1を足す

  • a++
  • ++a

aから1を引く

  • a--
  • --a

□ 演算子の優先順位

基本はひだりから
下記がある場合は優先順位に従って計算

  2. */%
  3. +-

型

  • 文字列 : String

    例）'こんばんは'　'test'　'0'

    ※ 補足
    シングルクオートで囲まれてる文字列内でシングルクオートを使いたい場合 : \'
    同様にダブルクオートで囲まれてる文字列内でダブルクオートを使いたい場合 : \"
    文字列を開業したい場合 : \n
  • 数値 : Number

    例）0　1

  • 論理値 : Boolean

    例）true　false

□ 型の変換

  • String();　文字列に変換
  • Number();　数字に変換
  • Boolean();　論理型に変換

例）String(10) + 10;　→　1010
Number('10') + 10;　→ 20

比較演算子

  • a == b　値が等しい場合にtrue
  • a === b　値と型が等しい場合 true
  • a != b 値が等しくない場合 true
  • a !== b　値と型が等しくない場合 true
  • a > b　a より b が小さい場合 true
  • a < b　a が b より小さい場合 true
  • a >= b　a が b 以上の場合 true
  • a <= b　a が b 以下の場合 true

論理演算子

  • a && b　a b 2つの式・値が正しい（tureの）場合 true
  • a || b　a b どちらかの式・値が正しい（tureの）場合 true
  • !a　aが正しくない（falseの）場合 true

jsのコメントの書き方

  • 一行コメント : //
  • 複数行コメント : /* */
