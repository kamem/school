a = b;
a に b を代入
a += b;
a = a + b
a -= b;
a = a - b
a *= b;
a = a * b

aに1を足す

a++;
a = a + 1
++a;
a = a + 1

aから1を引く

a--;
a = a - 1
--a;
a = a - 1

型

  • 文字列 : String

    例）'こんばんは', 'test', '0'

    ※ 補足
    シングルクオートで囲まれてる文字列内でシングルクオートを使いたい場合 : '\''
    同様にダブルクオートで囲まれてる文字列内でダブルクオートを使いたい場合 : "\""
    文字列を開業したい場合 : '\n'
  • 数値 : Number

    例）0 1

  • 論理値 : Boolean

    例）true false

□ 型の変換

  • String(); 文字列に変換
  • Number(); 数字に変換
  • Boolean(); 論理型に変換

例）String(10) + 10; → 1010
Number('10') + 10; → 20

比較演算子

a === b
値と型が等しい場合 true
a !== b
値と型が等しくない場合 true
a > b
a が b より大きい場合 true
a < b
a が b より小さい場合 true
a >= b
a が b 以上の場合 true
a <= b
a が b 以下の場合 true

論理演算子

a && b
aとb 2つの式・値がどちらも正しい場合 true
a || b
aとb どちらかの式・値が正しい場合 true
!a
値を反転する（aが正しくない場合 true、正しい場合はfalse）

例)

var num = 10;
//numが3より大きいかつ、20より小さい値のためtrue
console.log(num > 3 && num < 20)
// numが10もしくは20の場合trueなのでこの場合10 === 10がのためtrue
console.log(num === 10 || num === 20)
// 上と同じ値を!(式)!で反転しているためfalse
console.log(!(num === 10 || num === 20)) //false
