タブを作ってみる。

ページが読み込まれたら関数を実行

window.onload = load;

function load() {
処理
}

タグへのアクセス

getElementById(アクセスしたいタグのid);

idでタグアクセス

例

<div id="tab">
.
.
</div>
var tab = document.getElementById('tab');

getElementsByClassName(アクセスしたいタグのclass);

classでタグにアクセス（上から順番にクラスが付いているタグが入った、配列を返してくれます。）

例

<ul class="nav">
.
.
</ul>
//ページ全体のnavにアクセス
var nav = document.getElementsByClassName('nav');

//idタブの中のnavにアクセスしたい場合
var tab = document.getElementById('tab');
var nav = tab.getElementsByClassName('nav');

//配列が帰ってくるので最初のタグにアクセスしたい場合は
var tab = document.getElementById('tab');
var nav = tab.getElementsByClassName('nav')[0];

getElementsByTagName(アクセスしたいタグ名);

タグ名でアクセス（上から順番に指定したタグが入った、配列を返してくれます。）

<ul class="nav">
    <li id="nav0">コンテンツ1</li>
    <li id="nav1">コンテンツ2</li>
    <li id="nav2">コンテンツ3</li>
</ul>
//idタブの中のnavにアクセス
var tab = document.getElementById('tab');
// .navの一番最初のタグを取ってくる
var nav = tab.getElementsByClassName('nav')[0];
// .nav li を取ってくる
var navBtn = nav.getElementsByTagName('li');

クリック時に関数を実行

タグ.click = 実行したい関数;

例

document.onclick = test;

function test() {
	console.log('test');
}
