配列（Array）と連想配列（Object）

配列（Array）

配列の宣言

var fruits = [];
fruits  = ['りんご','バナナ','パイナップル','ぶどう','すいか','めろん'];

配列の取り出し方

変数名[配列番号]

※最初の配列を0番目として数えていくことに注意。

例）

console.log(fruits[0]); //りんご
console.log(fruits[1]); //バナナ
console.log(fruits[2]); //パイナップル
console.log(fruits[3]); //ぶどう
console.log(fruits[4]); //すいか
console.log(fruits[5]); //めろん

多次元配列

var fruits = [
	['りんご','さくらんぼ','いちご'], //赤い果物
	['かき','みかん','びわ'], // オレンジ色果物
	['バナナ','パイナップル','レモン'] //黄色い果物
];

多次元配列の取り出し方

変数名[配列番号][配列番号]

※最初の配列を0番目として数えていくことに注意。

例）

console.log(fruits[0][0]); //りんご
console.log(fruits[1][1]); //みかん
console.log(fruits[2][0]); //バナナ

連想配列（Object）

連想配列の宣言

var fruits = {};
fruits = {
	red : 'りんご',
	orange : 'みかん',
	yellow : 'バナナ'
};

※ 左側のred,orange,yellowの部分を 添字といいます。

連想配列の取り出し方

  1. 変数名.添字
  2. 変数名['添字']　※ こっちだと'添字'部分を変数にできるので 、うまい具合に使い分けましょう。

例）

console.log(fruits.red); //りんご
console.log(fruits['orange']); //みかん

var banana = 'yellow'
console.log(fruits[banana]); //バナナ

連想配列と多次元配列を使って整理

var fruits = {
	red : ['りんご','さくらんぼ','いちご'], //赤い果物
	orange : ['かき','みかん','びわ'], // オレンジ色果物
	yellow : ['バナナ','パイナップル','レモン'] //黄色い果物
};

取り出し方

console.log(fruits.red[0]); //りんご
console.log(fruits['orange'][1]); //みかん

var banana = 'yellow'
console.log(fruits[banana][0]); //バナナ
