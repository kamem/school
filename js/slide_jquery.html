CSSを変更する

$("タグ").css({ 
    css : 値
});

例

$('#slide .section').css({ 
    opacity : 0,
    marginLeft : 10 + 'px',
    left: 100 + 'px'
});

アニメーションさせる

$("タグ").animate({ 
    css : 値
}, スピード,イージング,アニメーションが終わった後に実行する関数 );

例

$('#slide .section').animate({ 
    opacity : 0,
    marginLeft : 10 + 'px',
    left: 100 + 'px'
}, 1000);

タグの数だけ繰り返し処理を行う

$("タグ").each(function(i){
処理
});

例

$('#slide .section').each(function(i){
	console.log(i);
	
	$(this).css({
		opacity : 0
	});
});
